// turbo-all

## Common Development Tasks

This workflow enables auto-confirmation for common development commands.

### Starting Development Server
```bash
npm run dev
```

### Building for Production
```bash
npm run build
```

### Installing Dependencies
```bash
npm install
```

### Git Operations
```bash
git status
git add .
git commit -m "message"
git push
```

### Linting and Formatting
```bash
npm run lint
npm run format
```

### Viewing Logs
```bash
npm run dev 2>&1 | head -100
```
